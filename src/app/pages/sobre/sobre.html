Sobre o Sydgames

Bem-vindo ao Sydgames, a sua plataforma para descobrir e explorar o vasto universo dos videogames. Nossa missão é simples: ajudar você a encontrar seu próximo jogo favorito, seja ele um grande lançamento ou uma joia independente.

Como Funciona?

Utilizamos uma interface limpa e rápida para que você possa pesquisar em um catálogo com milhares de títulos. Basta digitar o nome de um jogo na barra de pesquisa para ver informações detalhadas, incluindo descrições, data de lançamento, avaliações e capturas de tela.

A Tecnologia por Trás

Para fornecer dados atualizados e abrangentes, nosso site é orgulhosamente desenvolvido utilizando a API RAWG (rawg.io), a maior base de dados de videogames da internet. Agradecemos à equipe da RAWG por disponibilizar uma ferramenta tão poderosa para a comunidade de desenvolvedores.

O front-end foi construído com Angular, um dos mais modernos e eficientes frameworks para aplicações web.

Quem Desenvolveu?

Este projeto foi criado por um entusiasta de desenvolvimento de software e, claro, um apaixonado por games. O Sydgames nasceu da vontade de aplicar conhecimentos técnicos em um tema divertido e criar uma ferramenta útil para outros jogadores.
